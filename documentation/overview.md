Goal:
The goal of this project is to write a C shell from scratch. Learning new skills, tools, and how to interface with hardware on a lower level. The secondary goal is to implement and leverage AI into the project, specifically learning how to interface with AI and use it to increase productivity.

Technologies Used:
WSL has been used for testing and development, VIM as a text editor. For the last AI feature VirtualBox will be used for the safety of a virtual machine.
Git for version control software.
Ollama which is a free tool which allows for local use of LLM's for AI functionality.
Flex and Bison, a lexical analyzer and parser respectively.
Extensive use of Stack Overflow and Googles AI mode for troubleshooting and debugging complex errors.
